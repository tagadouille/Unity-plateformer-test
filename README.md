# 🎮 Unity Platformer Prototype

Ce projet est un prototype de jeu de plateforme inspiré des classiques du genre, développé avec **Unity**. Il a été conçu comme un terrain d’expérimentation pour explorer et maîtriser les bases du développement de jeux vidéo.

## ✨ Fonctionnalités principales

Le prototype inclut déjà plusieurs mécaniques essentielles :

* 🕹️ **Déplacement du joueur** fluide
* 🎥 **Suivi de caméra**
* 🧱 **Gestion des collisions**
* 👾 **Ennemis basiques**
* 🔄 **Plateformes dynamiques** (déplacement et rotation sur les axes X, Y, Z)

Ces éléments constituent une base solide pour la création de niveaux plus complexes et variés à l’avenir.

## 🎯 Objectif du projet

Ce projet s’inscrit dans une démarche d’**auto-formation** sur **Unity**.
Il vise à acquérir les compétences nécessaires pour développer des jeux plus ambitieux et complets.

## ⚙️ Prérequis

⚠️ Aucun exécutable n’est disponible pour le moment.

Pour lancer le projet, vous devez :

* Installer **Unity 6**
* Ouvrir le projet dans Unity
* Charger la scène **`MainMenu`**
* Lancer le jeu via l’onglet **Game**

> 💡 Un build exécutable sera ajouté dans les versions futures du projet.

## 🎮 Contrôles

* **ZQSD** : déplacer le personnage
* **Espace** : sauter

## 🏁 Objectif du jeu

Le but est simple : **atteindre la ligne d’arrivée** 🏁

Pour y parvenir, le joueur devra éviter différents obstacles :

* Plateformes fixes et mobiles
* Plateformes tournoyantes
* Ennemis

## 🚀 Perspectives d’évolution

* Ajout de nouveaux niveaux
* Amélioration des mécaniques de gameplay
* Système de score ou de progression
* Création d’un build jouable (exécutable)
